# Banking-System-Using-Python
A simple Python-based Banking System that supports secure account login, balance inquiry, deposits, and withdrawals. Includes credential validation, error handling, and real‑time balance updates. Designed to demonstrate core Python skills and logical problem‑solving.
## 📌 Features
## 🔐 Account Login
Users can log in using an account number/username and PIN/password

Validates credentials before granting access

Displays clear error messages for invalid login attempts

## 💰 Deposit Money
Allows logged‑in users to deposit funds

Ensures deposit amount is greater than zero

Updates and displays the new account balance

## 🏧 Withdraw Money
Allows users to withdraw funds

Validates that withdrawal amount is positive

Prevents withdrawals exceeding available balance

Shows updated balance after withdrawal

## 🎯 Project Objective
This mini‑project evaluates:

Python coding skills

Logical thinking

Ability to implement real‑world operations

Understanding of input validation and error handling

## 🛠️ Tech Stack
Python 3

Console‑based user interaction
